Greedy

Posted on 2019-10-03

Strategy

A greedy algorithm always makes a local optimal choice to explores a global optimal solution.

When we are using greedy algorithm, we shall always consider which local optimality we shall take as the measure (or objective function), and prove such local optimality will leads to the global optimality.

Activity Selection Problem

Definition

Let \(S\) be a set of activities with cardinality \(n\), which is \(S = \{ a_i : 1 \leq i \leq n \}\). Each activity \(a_i\) is an half-open interval with a start time \(s_i\) and a end time \(f_i\), which is \(a_i = [s_i, f_i)\). Activity \(a_i\) and \(a_j\) are compatible if \(a_i \cap a_j = \phi\).

Local Optimality

One of the local optimality is that always take the smallest finishing time into solution sets.

The complete algorithm is as follows.

  1. Add the interval \(I \in S\) with the smallest finish solution set \(T\).
  2. Remove all intervals that intersecting with \(I\) from \(S\).

The time complexity of this algorithm is \(\Omega(n \log n)\).

Correctness

Let \(\left | S \right | = n\).

When \(n = 1\), the optimal solution is \(T = S\) and it will contains the interval with smallest finishing time. The algorithm is correct.

We claim that if \(I = [s, f]\) be the interval with the smallest finishing time, then there exists a optimal solution set \(T\) which contains \(I\).

Suppose \(T^*\) is an arbitrary solution set that does not contain \(I\). We can always convert \(T^*\) into solution set that contains \(I\) with same cardinality.

Let \(I^* = [s^*, f^*]\) be the interval with the smallest finishing time in \(T^*\), that is, for all interval \(J_i \in T^* / \{ I^* \}, 1 \leq i \leq \left | T^* \right | - 1\), we have \(s_i \ leq f^*\).

Let \(T = (T^* / \{ I^* \}) \cup \{ I \}\), since \(s \leq f < f^* \leq s_i\), \(I\) is also disjoint with all \(J_i\) and \(\left | T \right | = \left | T^* \right |\). Therefore, \(T\) is also an optimal solution for the problem.

Assume that the algorithm is correct for all \(n \leq k\).

When \(n = k + 1\), Let \(I\) be the interval with the smallest finishing time in \(S\) and \(T^*\) be the optimal solution of the \(S^* = \{ J : I \cap J = \phi, J \in S \}\). We will prove that \(T^* \cup \{ I \}\) is an optimal solution by contradiction.

Suppose not, then there exists a solution set \(T\) such that \(T \in I\) as the result of the above lemma. Since \(T^* \cup \{ I \}\) is not optimal, we have \(| T^* \cup \{ I \} | < | T |\).

On the other hand, since \(T / \{ I \}\) is a subset of \(S^*\) since every element in \(T / \{ I \}\) is disjoint with \(I\) and \(|T^*|\) is an optimal solution of \(S^*\). We have \(|T^*| \geq |T / \{ I \}|\) and thus \(|T^* \cup \{ I \}| \geq |T|\). Contradiction arises.
