Hexo - 快速開始

Posted on 2019-08-23

安裝

依賴組件

Hexo需要使用npm和git來安裝管理其依賴組件，所以請確保先安裝以下軟件： * Node.js >=6.9 * Git 在安裝Node.js和Git之後，於命令行輸入以下指令就能檢視當前版本。

安裝 Hexo

使用npm即可以下載Hexo CLI。

然後在指定位置($folder)創建新的博客系統。

配置

_config.yml _config.yml就是Hexo内建的配置文件，可以參考Hexo 配置來獲取完整的配置清單和簡介。常用配置項目如下表所示：

網站

設定 描述
title 網站標題，顯示於導航列和網頁分頁標題
subtitle 網站副標題，顯示於導航列
author 您的名字
language 網站預設的語言
timezone 網站預設的時區

網址

設定 描述
url 博客所佈署的網址
root 博客所佈署的根目錄
permalink 永久鏈結的格式
